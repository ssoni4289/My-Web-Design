
  HTML : hypertext markup language , it has collections of pre defined tags.
  it is used to design basic structure of webpage.
  
  Tag : <>  - open tag
        </>  - close tag
		
	Type of tags - 
	   1. input tags - ask input from user
	   2. output tags - display output to user
	   
	Behavior of html tags - 
	   1. dependent tags - Those tags, that has open and close.
	                       eg - <h1> </h1>
						        <body> </body>
	   2. independent tags - those tags that has only open.
	                       eg - <br/>
						        <hr/>
								<img/>
	   3. block level tags - those tags print statement in new line 
       4. inline tags - print statement in same line by default.
	   
	   Note - responsive design - using css block level tag => inline
	                              using css inline tag => block level

		how to link pages in html.
			 - <a href="nextfilename" target="" title=""> text to link </a>
			 
				example :
					<a href="login.html" target="_new" title="click here"> click to go </a>
							
					login.html
					register.html

				About properties of a tag in html - 
					href: it need next filename to redirect
					title: it is for user guidance
					target: it define the location for next file to open
						type of target - 
							_self - it open next file in same tab.
							_new - it open next file in new tab but only at first click.
							_blank: it open next file in new tab but on every click.
							
							using frameset - 
							_top : it link to master(main) page
							userDefine : we can create our own target also
							
							using javascript -
							_child: it open a popup window
							_parent: will clear once start javascript
