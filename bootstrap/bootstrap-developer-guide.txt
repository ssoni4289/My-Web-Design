Bootstrap is library of css. It is also the first framework 
of html, css, java script for resposive web page designing.

Bootstrap is introduced by twitter.com in 2011.
Bootstrap has pre-defined css classes for each design.
Bootstrap is designed based on grid system.

Grid system has devided all the devices in 5 types.
1. Extra small :- 0px - 576px
2. Small :- 577px - 768px
3. Medium :- 769px - 992px
4. Large :- 993px - 1200px
5. Extra large :- greater than 1200px

While designing a web page using bootstrap follow these 
steps - 

1. In head section, link to bootstrap.css file.
2. Follow the body tag.
3. Inside body follow these classes.
	a. container - it has more margin from left and right
	b. container-fluid - it has less margin from left and right
4. Inside the container follow the class
	a. row
5. Inside the row, follow the columns class from given 60
	
Final structure - body -> container -> row -> columns(1 to 16) -> page contents	
	
Note: Each row follow min one cloumn and max 12 columns.
