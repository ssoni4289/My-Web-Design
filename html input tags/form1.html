List of properties for single line input elements.

text, password, email, number

Placeholder : it is used to guide the user for input.

Maxlength : it is used to set the limit for input elements in char in text, email, password

size : it is use to resize the width of text, email, password.

value : it is use to set the default value for text, email, password, number

tabindex : it is use to set sequence of cursor while pressing tab key from computer

readonly : it will make input element readonly

disabled : it disable (non functional) the element , it can not communicate to backend.

name : it use to collect input from user and pass to backend

id : it is use to communicate with css, javascript , ajax, jquery.

class : it is use to communicate with css, javascript , ajax, jquery
